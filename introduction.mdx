## What is Motion Start?

Motion Start is a production-ready animation library for Svelte 5, inspired by the popular React library Framer Motion. It provides a simple yet powerful API for creating fluid animations, gesture-based interactions, and complex motion sequences in your Svelte applications.

Built on top of proven animation primitives from Popmotion and Framesync, Motion Start brings the best of Framer Motion's declarative animation API to the Svelte ecosystem.

## Key features

Motion Start provides a comprehensive set of animation capabilities:

## Simple example

Here's a basic example showing how to create an animated button with hover effects:

```svelte
<script>
  import { Motion } from "motion-start";
</script>

<Motion.div
  whileHover={{
    scale: 0.8,
    rotate: 45,
  }}
  class="box"
>
  Hover me
</Motion.div>
```

This creates an interactive element that scales and rotates when you hover over it, with smooth animations handled automatically.

## Spring animations

Motion Start uses spring physics for natural-feeling animations:

```svelte
<script>
  import { Motion } from "motion-start";
</script>

<Motion.div
  animate={{
    rotate: 180,
  }}
  transition={{
    type: "spring",
    damping: 10,
    mass: 0.75,
    stiffness: 100,
  }}
  class="box"
/>
```

This guide will walk you through creating your first animations with Motion Start. You'll learn how to animate elements, respond to user interactions, and create smooth spring-based animations.

## Prerequisites

Before you begin, make sure you have:

- A Svelte 5 project set up
- Motion Start installed (see [Installation](/installation))

## Your first animation

<Steps>
  <Step title="Import Motion component">
    First, import the `Motion` component from Motion Start:

    ```svelte title="src/routes/+page.svelte"
    <script>
      import { Motion } from "motion-start";
    </script>
    ```

    The `Motion` component provides access to animated versions of all HTML elements through a proxy (e.g., `Motion.div`, `Motion.button`).
  </Step>

  <Step title="Create an animated element">
    Use `Motion.div` to create an animated element with the `animate` prop:

    ```svelte title="src/routes/+page.svelte"
    <script>
      import { Motion } from "motion-start";
    </script>

    <Motion.div
      animate={{
        rotate: 360,
      }}
      transition={{
        duration: 2,
      }}
      class="box"
    >
      Watch me spin!
    </Motion.div>

    <style>
      .box {
        width: 150px;
        height: 150px;
        background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
        border-radius: 12px;
        display: flex;
        align-items: center;
        justify-content: center;
        color: white;
        font-weight: bold;
      }
    </style>
    ```

    This creates a rotating box that animates to 360 degrees over 2 seconds.
  </Step>

  <Step title="Add interactive gestures">
    Make your animation interactive with gesture props like `whileHover` and `whileTap`:

    ```svelte title="src/routes/+page.svelte"
    <script>
      import { Motion } from "motion-start";
    </script>

    <Motion.div
      whileHover={{
        scale: 0.86,
      }}
      whileTap={{
        scale: 0.9,
        rotate: 45,
      }}
      class="box"
    >
      Hover and click me!
    </Motion.div>

    <style>
      .box {
        width: 150px;
        height: 150px;
        background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
        border-radius: 12px;
        display: flex;
        align-items: center;
        justify-content: center;
        color: white;
        font-weight: bold;
        cursor: pointer;
      }
    </style>
    ```

    <Info>
      The element automatically animates back to its original state when you stop hovering or release the click.
    </Info>
  </Step>

  <Step title="Use spring physics">
    Replace the default tween animation with spring physics for more natural motion:

    ```svelte title="src/routes/+page.svelte"
    <script>
      import { Motion } from "motion-start";
    </script>

    <Motion.div
      animate={{
        rotate: 180,
      }}
      transition={{
        type: "spring",
        damping: 10,
        mass: 0.75,
        stiffness: 100,
      }}
      class="box"
    >
      Spring animation!
    </Motion.div>

    <style>
      .box {
        width: 150px;
        height: 150px;
        background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
        border-radius: 12px;
        display: flex;
        align-items: center;
        justify-content: center;
        color: white;
        font-weight: bold;
      }
    </style>
    ```

    Spring parameters:
    - **type**: Set to `"spring"` to use spring physics
    - **damping**: Controls the friction (lower = more bouncy)
    - **mass**: The mass of the object (affects momentum)
    - **stiffness**: The spring strength (higher = faster)
  </Step>
</Steps>

## Complete example

Here's a complete example combining multiple concepts:

```svelte title="src/routes/+page.svelte"
<script>
  import { Motion } from "motion-start";
  let count = 0;
</script>

<div class="container">
  <h1>Motion Start Demo</h1>
  
  <div class="boxes">
    <!-- Tween animation -->
    {#key count}
      <Motion.div
        animate={{ rotate: 360 }}
        transition={{ duration: 2 }}
        class="box"
      >
        Tween
      </Motion.div>
    {/key}

    <!-- Spring animation -->
    {#key count}
      <Motion.div
        animate={{ rotate: 180 }}
        transition={{
          type: "spring",
          damping: 10,
          mass: 0.75,
          stiffness: 100,
        }}
        class="box"
      >
        Spring
      </Motion.div>
    {/key}

    <!-- Interactive gestures -->
    <Motion.div
      whileHover={{ scale: 0.8, rotate: 45 }}
      whileTap={{ scale: 0.9 }}
      class="box"
    >
      Interactive
    </Motion.div>
  </div>

  <button onclick={() => count++}>
    Restart animations
  </button>
</div>

<style>
  .container {
    max-width: 800px;
    margin: 0 auto;
    padding: 2rem;
    text-align: center;
  }

  h1 {
    margin-bottom: 2rem;
  }

  .boxes {
    display: flex;
    gap: 2rem;
    justify-content: center;
    margin-bottom: 2rem;
    flex-wrap: wrap;
  }

  .box {
    width: 150px;
    height: 150px;
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    border-radius: 12px;
    display: flex;
    align-items: center;
    justify-content: center;
    color: white;
    font-weight: bold;
    cursor: pointer;
  }

  button {
    padding: 0.75rem 1.5rem;
    background: #667eea;
    color: white;
    border: none;
    border-radius: 8px;
    font-weight: bold;
    cursor: pointer;
    transition: background 0.2s;
  }

  button:hover {
    background: #764ba2;
  }
</style>
```

## Result

You should now see three animated boxes:

1. **Tween**: Rotates 360° with a linear duration-based animation
2. **Spring**: Rotates 180° with physics-based spring animation
3. **Interactive**: Scales and rotates on hover and tap

Clicking "Restart animations" re-triggers the animations using Svelte's `{#key}` block.

## Animation types

Motion Start supports two main animation types:

### Tween (duration-based)

```typescript
transition: {
  duration: 2,        // Animation duration in seconds
  ease: "easeInOut",  // Easing function
  delay: 0.5,         // Delay before starting
}
```

### Spring (physics-based)

```typescript
transition: {
  type: "spring",
  stiffness: 100,     // Spring strength
  damping: 10,        // Friction/resistance
  mass: 0.75,         // Object mass
}
```

<Tip>
  Spring animations feel more natural for interactive UI elements. Use tween animations when you need precise timing control.
</Tip>
